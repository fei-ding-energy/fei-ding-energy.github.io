# fei-ding-modern-site

A modern, dark, responsive personal site for GitHub Pages (no external Jekyll theme required).

## Deploy
1. Create a new GitHub repo and upload these files.
2. Go to **Settings â†’ Pages** and select **GitHub Actions** or **Deploy from a branch**.
3. Visit your site URL when the build completes.

## Customize
- Edit `index.md` and page markdown under each folder.
- Update `_config.yml` `url` and social links.
- Add images to `assets/` and reference them in Markdown.
